# Socket.IO Chat Application

This project implements a real-time chat system using Socket.IO and Axios for handling backend communication. It supports user authentication, message sending, fetching messages, and handling room-based interactions.

## Features

- **Authentication**: Verifies users by checking their tokens.
- **Real-time Messaging**: Allows users to send and receive messages in real time.
- **Room-Based Communication**: Users can join specific rooms for private or group conversations.
- **Message History**: Retrieve past messages from a room.
- **Error Handling**: Notifies users of any errors that occur during interactions.

## Requirements

- **Node.js**: The application is built using Node.js.
- **Socket.IO**: For managing real-time communication.
- **Axios**: For making HTTP requests to the backend API.
- **dotenv**: For handling environment variables.
